# universal-formating

Format data to desired output format.

## Features
- Csv to markdown table;
- Markdown table to csv (WIP);
- WIP [...].

## CSV to MARKDOWN TABLE Usability

Put the csv in input textarea

```csv
name;age;hobby
Anderson;25;Play games
```

Get the output in markdown table

```markdown
| name     | age | hobby      |
|----------|-----|------------|
| Anderson | 25  | Play games |
```
| name     | age | hobby      |
|----------|-----|------------|
| Anderson | 25  | Play games |
